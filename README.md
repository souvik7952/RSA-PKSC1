# RSA-PKSC1
**RSA Cryptographic Tool** is a fully client-side browser-based utility for RSA/ECB/PKCS1 encryption and decryption. It supports RSA key pair generation (1024/2048/4096-bit), PEM key import/export, Base64/Hex encoding, PKCS#1 v1.5 padding, and secure encryption/decryption with a modern dark/light mode interface.
